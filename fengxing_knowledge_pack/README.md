# 丰行慧运知识包 v0.2

> 用途：供 Codex 在公开互联网中自主发现、筛选、核验并分析与丰行慧运相关的情报。
>
> 状态：业务理解增强版。基于 2026 年大会介绍、经营会议材料、货主业务产品策略规划，以及丰图历史交通货运、安全风控、监管、智慧交通和泛零售方案整理而成，包含“当前材料支持”“历史能力参考”“推断/待确认”三种信息等级。

## 使用原则

1. **丰行与丰图分开处理。** 本知识包只定义丰行慧运的交通货运业务边界。丰图科技材料仅作为技术与项目经验的历史参考，不能自动等同于丰行当前可售能力。
2. **当前战略优先使用 2026 年材料。** 2022-2025 年资料主要用于理解能力演进、方法论和历史案例。
3. **不得把内部计划写成已实现事实。** 2026 年目标、规划、商机、试点及收入数字均属于内部规划，除非有公开来源再次确认。
4. **对外情报必须引用公开互联网来源。** 本知识包只用于判断“什么与丰行有关”和“如何分析”，不能作为外部事件事实来源。
5. **资料冲突时使用更晚版本，并标记冲突。**

## 目录

- `01_company_profile.md`：丰行定位、边界与价值主张
- `02_business_map.md`：业务版图与上下游角色
- `03_product_capability_matrix.md`：产品与能力矩阵
- `04_customer_scenarios.md`：客户、场景与痛点
- `05_strategy_2026.md`：2026 年业务战略
- `06_intelligence_scope.md`：情报研究范围、入选标准与排除项
- `10_claims_and_sources.md`：事实等级与来源索引
- `11_pending_confirmation.md`：需要业务负责人确认的事项
- `12_business_understanding.md`：丰行业务深度理解总纲
- `knowledge.json`：机器可读知识摘要

长期执行规则以仓库根目录 `AGENTS.md` 为准；观察名单、查询库和实体别名分别以 `config/watchlist.yaml`、`config/query-library.yaml`、`config/entity-aliases.yaml` 为准。知识包 v0.2 中同类旧文件已移入 `legacy/knowledge-pack-config-v0.2/`，仅供历史追溯。
