## 使用规则

- 区分当前正式能力、已交付或验证能力、试点能力、规划能力、历史能力和待确认能力。
- 对规划中的产品，只能使用“规划、探索、拟、可能、待确认”等表述。
- 不得把丰图通用能力直接写成丰行现有产品，除非知识包明确确认归属。
