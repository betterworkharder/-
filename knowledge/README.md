# 丰行慧运知识入口

本目录是项目规则中 `knowledge/` 的稳定入口。正式研究前先读本文件，再按任务需要进入分层目录。

## 当前有效知识入口

- `fengxing_knowledge_pack/`
- `knowledge/current/`
- `knowledge/product-strategy/`
- `knowledge/historical/`
- `knowledge/source-extracts/`

## 推荐读取顺序

1. `fengxing_knowledge_pack/README.md`
2. `fengxing_knowledge_pack/knowledge.json`
3. `fengxing_knowledge_pack/10_claims_and_sources.md`
4. `fengxing_knowledge_pack/11_pending_confirmation.md`
5. `knowledge/current/README.md`
6. `knowledge/product-strategy/README.md`
7. `knowledge/historical/README.md`

再按任务需要读取 `knowledge/source-extracts/` 中的抽取文本。

## 重要边界

- `knowledge/source-extracts/` 来自业务文档抽取文本，包含历史项目、内部经营数字、客户信息或规划内容，不得直接写入 `google-ai-studio-input.md`。
- 历史材料只能用于理解能力来源、方法论和搜索方向，不能证明当前产品仍在售、历史客户仍合作或历史项目已转化为当前规模化业务。
